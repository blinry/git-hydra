NodeFactory
-----------

Wird nach einem Objekt gefragt, liest aus einem Git-Repository die notwendigen Informationen zum Erzeugen eines Nodes mit seinen Verknüpfungen aus und gibt sie zurück.

Graph
-----

Repräsentiert momentane "Ansicht" des Repositorys. Enthält eine Menge von Knoten, die mit gerichteten, beschrifteten Kanten miteinander verbunden sind, ihre räumliche Anordnung, ob sie angezeigt werden, ob sie vollständig sind und ihren "Typ". Soll ein Knoten ausgeklappt werden, der noch nicht vollständig ist, wird die verbundene NodeFactory zu Rate gezogen.

Node
----

Hat eine Position, eine Beschriftung, eine ID und eine Liste von Kanten.

Edge
----

Paar aus Knoten-ID und Kantenbeschriftung.

ForceDirectedLayout
-------------------

Layoutet Graph anhand von Hookes und Coulomb's law.

SFMLDisplay
-----------

Zeichnet einen Graphen mit Hilfe von SFML.

Git-Tutor
---------

Erzeugt einen Graphen und weist ihm eine NodeFactory zu.
Erzeugt ein Layout. Erzeugt ein Display.
Geht in eine Endlosschleife, lässt Graph layouten und anzeigen. Außerdem das Display seine Events verarbeiten. Wenn das Display meldet, dass es geschlossen wurde, Programm beenden.
